# FastAPI JSON File Server

This project implements a simple FastAPI server that serves a JSON file and provides a status endpoint.

## Features

- Serves a JSON file.
- Provides a server status endpoint.

## Requirements

- Python 3.7 or higher
- FastAPI
- Uvicorn
